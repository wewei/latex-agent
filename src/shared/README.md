# 共享代码 (Shared Code)

共享代码目录包含可以在主进程、预加载脚本和渲染进程之间共享的代码。

## 主要职责

- 提供共享的类型定义
- 定义共享常量
- 提供共享工具函数
- 确保类型安全

## 开发规范

- 使用 ADT 定义类型
- 保持代码的纯函数性质
- 避免进程特定依赖
- 确保类型安全

## 目录结构

- `types/` - 共享的类型定义
  - `ipc.ts` - IPC 消息类型
  - `state.ts` - 共享状态类型
- `constants/` - 共享常量
  - `events.ts` - 事件名称常量
- `utils/` - 共享工具函数
  - `validation.ts` - 数据验证函数

## 使用规范

1. 类型定义：
   - 使用 ADT 定义共享类型
   - 确保类型定义不依赖进程特定代码
   - 通过 index.ts 统一导出类型

2. 常量定义：
   - 使用 const 断言确保类型安全
   - 统一管理事件名称等常量
   - 避免魔法字符串

3. 工具函数：
   - 保持纯函数性质
   - 避免副作用
   - 使用 ADT 处理错误和异步
