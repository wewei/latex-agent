# 主进程 (Main Process)

主进程是 Electron 应用的核心，负责管理应用的生命周期和系统级操作。

## 主要职责

- 应用生命周期管理（启动、退出等）
- 系统级 API 调用（文件系统、系统托盘等）
- 窗口管理（创建、销毁窗口）
- IPC 通信（与渲染进程通信）

## 开发规范

- 使用函数式编程范式
- 使用 ADT 定义 IPC 消息类型
- 避免全局状态
- 保持代码的纯函数性质

## 文件组织

- `index.ts` - 主进程入口文件
- `ipc.ts` - IPC 通信处理
- `window.ts` - 窗口管理
- `app.ts` - 应用生命周期管理
